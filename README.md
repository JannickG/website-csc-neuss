# CSC Neuss Website – Vorschau & Feedback

Dieses Projekt enthält die aktuelle Entwicklungs­version der Website.



### Lokal Die Seite öffnen:
---

## 1. Node.js installieren (einmalig)

Falls noch nicht installiert:

1. https://nodejs.org öffnen  
2. **LTS-Version** herunterladen  
3. Mit Standardeinstellungen installieren  
4. Rechner neu starten  

---

## 2. Projektordner öffnen

- ZIP-Datei entpacken (falls als ZIP erhalten)
- Projektordner öffnen

---

## 3. Terminal im Projektordner öffnen

### Windows
- Ordner öffnen  
- In die Adressleiste klicken  
- `cmd` eingeben  
- Enter drücken  

### Mac
- Ordner öffnen  
- Rechtsklick im Ordner  
- „Neues Terminal im Ordner“

---

## 4. Abhängigkeiten installieren

Im Terminal eingeben:

```
npm install
```

Warten, bis der Vorgang abgeschlossen ist.

---

## 5. Website starten

Im Terminal eingeben:

```
npm run dev
```

---

## 6. Im Browser öffnen

Im Browser aufrufen:

```
http://localhost:3000
```

---

## Server stoppen

Im Terminal:

```
CTRL + C
```

---

## Feedback

Feedback gerne direkt weitergeben.  
Anpassungen werden schnellstmöglich umgesetzt.
